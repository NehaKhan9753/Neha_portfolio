    Projects

    AI-Driven Event Scheduling Agent

    AI-powered scheduling assistant that I developed, designed to revolutionize event management. Leveraging OpenAI API for Natural Language Processing (NLP), the system intelligently processes event details from both text and voice messages, offering unparalleled flexibility. It seamlessly integrates with Telegram Bot API for user interaction and Google Calendar API for automated event creation. This innovative solution, built using n8n for workflow automation, ensures efficient and seamless event scheduling, enhancing productivity and user experience.

    Full-stack E-commerce web application

    I developed a complete e-commerce website from front to back, using React for a smooth and interactive user experience. On the backend, I built a strong API with Node.js and Express, storing data securely in MongoDB Atlas. This platform includes all the essentials for online shopping: secure logins, easy product Browse and filtering, a shopping cart, and payment processing. Plus, it has separate dashboards for both users and administrators to manage everything efficiently. It's a full-featured online store designed for a great shopping experience.

    Predicting S&P 500 Index Prices Using Machine Learning

    I built a machine learning model in Python designed to predict the future prices of the S&P 500 stock index. To make sure it was reliable, I rigorously tested it against past data (backtesting) and carefully analyzed the results to confirm its accuracy and effectiveness. This project demonstrates my ability to apply machine learning to real-world financial data for predictive analysis.
